# WhatsApp Automation

Este proyecto contiene un script de automatización para WhatsApp.

## Requisitos

- Python 3.x
- Instala las dependencias con: `pip install -r requirements.txt`

## Uso

Ejecuta el script MeAuto.py para automatizar tareas en WhatsApp.

## Instalación

1. Clona el repositorio.
2. Crea un entorno virtual: `python -m venv .venv`
3. Activa el entorno: `.venv\Scripts\activate` (Windows)
4. Instala dependencias: `pip install -r requirements.txt`
5. Ejecuta: `python MeAuto.py`
